Курсы для начинающих

Курсы английского для начинающих имеют ряд преимуществ:

  • Знакомство с азами письма и чтения на английском;
  • Базовые навыки для общения в повседневных ситуациях;
  • Поддержание диалогов на английском языке;
  • Восприятие разговорной речи на слух.

Как подтверждение нового уровня знаний, вам выдадут сертификат.

Продолжительность курса

Beginner - 60 академических часа

A1 - 144 академических часа

Описание учебного процесса

Перед началом занятий вам предложат пройти тест, который более точно определит для вас подходящую группу.

Вы сможете посетить открытый урок, на котором познакомитесь с преподавателем и оцените его уровень.

Учебные материалы

Программа New English File Beginner 2nd Edition, Oxford University Press;

Программа English File Elementary 3rd Edition, Oxford University Press;

Принципы коммуникативной методики:

  • Отсутствие языка-посредника с первого занятия
  • Естественная среда общения
  • Принцип ситуативности. Реальные языковые ситуации
  • Эффект присутствия в англоговорящей среде
  • Преподаватель не наставник, а компаньон
  • Акцент на использовании речевых оборотов, а не отдельных слов
  • Общение — это одновременно и цель, и средство

Учебные пособия

На занятиях используются учебные пособия English File, издания Oxford University Press. Пособие каждого уровня состоит из учебника и рабочей тетрадки.

Курс обучения для начинающих состоит из 2 уровней:

A1: English File Beginner 2nd Edition, Oxford University Press; 60 ак.ч.

A1: English File Elementary 3rd Edition, Oxford University Press; 144 ак.ч.
